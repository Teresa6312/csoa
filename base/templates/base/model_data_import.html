Import Model Data

Sample data format can be found in the reference folder
app(Django app label): Django app to import data for.
model (Model label): Django model class to update or create.
data (list of dict): Data to update the instance with. Keys may include:
- if all of this dict keys don't have "-" prefix, then the dict data is use to create new instance.
- "fieldname": Direct value for a field if it is not a model related field, like forgien key, many-to-many, etc.
- "-fieldname": Used in filters for get the unique instance.
